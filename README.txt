IMPORTANT INFORMATION:
1. The create commands are generated by django. The database schema are specified in form of models in the models.py file in the Complete Source Code/newApp file.

2. The files
 online_book_store-master/newApp/models.py	-> Definition of database schema
 online_book_store-master/newApp/views.py	-> Backend code to handle http requests and do sql queries
 online_book_store-master/newApp/urls.py	-> Mapping of html urls to the function in views.py
 online_book_store-master/newApp/forms.py	-> Input forms used in the html front end
 online_book_store-master/newApp/templates	-> Frontend code for the browser display

3. To run :
A. If only any change to database - models.py then 
	python manage.py makemigrations
	python manage.py makemigrations newApp
	python manage.py migrate

B. Go to directory  online_book_store-master/
	command : python manage.py runserver <port>
	Go to the browser and to the page http://127.0.0.1:<port>/
	To run admin console http://127.0.0.1:<port>/admin

4. Dependencies
A. Python 3.7.6
B. Django 3.0.4 

Rest of them in file online_book_store-master/requirements.txt
